# cat-mash

Cat-mash is a mini web application that allows you to find
the cutest cat.
<br>
Vote between 2 cat who is the cutest and watch the leaderboard whenever you want.

url : https://cat-mash.netlify.app/

## Project setup
```
npm install
```
